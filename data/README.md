**Paso a Paso para instalar los data frame en Kaggle**
1. Entrar en el sigiente link "https://www.kaggle.com/competitions/titanic"

2. En la parte superior entrar al apartado que dice "Data

3. Bajar hasta encontrar en el costado derecho el boton "Dowland all"

4. Encontrar la carpeta "titanic"

5. Extraer los archivos "Test" y "Train"
