# taller-pandas-titanic

**Descripción**

El proyecto consiste en un Análisis Exploratorio de Datos (EDA) con Pandas para determinar los factores de supervivencia del Titanic, abarcando la preparación de datos, el análisis estadístico profundo (incluyendo proporciones y agrupaciones complejas), y la síntesis de resultados en un informe. La entrega requiere un flujo de trabajo colaborativo estricto en GitHub Desktop, utilizando ramas, Pull Requests (PRs) con revisión, y la resolución documentada de conflictos.

**Objetivos**

Establecer un flujo de trabajo colaborativo eficaz utilizando la plataforma GitHub y la herramienta GitHub Desktop.
Desarrollar gráficos informativos que sinteticen los hallazgos clave, asegurando que cada visualización aporte una interpretación clara.
Generar estadísticas básicas y hacer comparaciones entre los conjuntos de datos de entrenamiento (train) y prueba (test).
